# Hound.Core — Shared Library

Shared models, interfaces, and abstractions used by all packs and the API.

## Key Types
- `Pack`, `HoundInfo`, `ActivityLog`, `HoundMessage` — Domain models in `Models/`
- `IActivityLogger` / `RavenActivityLogger` — Activity logging to RavenDB (in `Logging/`)
- `IOllamaClientFactory` / `OllamaClientFactory` — LLM client factory for Ollama (in `LlmClient/`)

## Patterns
- Interfaces live alongside their implementations
- Models are plain C# classes (no EF, no attributes — RavenDB is schemaless)
- Each pack gets its own RavenDB database for isolation
- `IActivityLogger` is the primary cross-cutting concern — every hound uses it

## Dependencies
- `RavenDB.Client` — Document store client
- `Microsoft.Agents.AI` — Agent Framework types
- `Microsoft.Extensions.Http` — `IHttpClientFactory`

## Do NOT add
- ASP.NET Core references (that's Hound.Api)
- Trading-specific types (that's Hound.Trading)
- UI dependencies
