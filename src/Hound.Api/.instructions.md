# Hound.Api — Monitoring API

ASP.NET Core REST API + SignalR hub for the monitoring dashboard.

## Endpoints
- `GET /api/packs` — List all packs
- `GET /api/packs/{id}` — Single pack detail
- `GET /api/packs/{packId}/hounds` — Hounds in a pack
- `GET /api/activity` — Paginated activity log (filters: pack, hound, from, to)

## SignalR Hub
- Endpoint: `/hubs/activity`
- Client methods: `SubscribeToPack(packId)`, `UnsubscribeFromPack(packId)`
- Server pushes: `OnActivity(ActivityLog)`

## Key Patterns
- Controllers are thin — delegate to RavenDB queries
- CORS configured for Angular dev server (`http://localhost:4200`)
- RavenDB indexes for efficient querying by pack/hound/time
- `IDocumentStore` injected via DI

## Dependencies
- `Hound.Core` — Models, `IActivityLogger`
- `RavenDB.Client` — Document store
- `Microsoft.AspNetCore.SignalR` — Real-time hub

## Do NOT
- Add trading logic here — this is read-only monitoring
- Return unbounded queries — always paginate
